# BiharCraftHub

BiharCraftHub is a multi-page front-end website created to showcase the rich cultural heritage and traditional handicrafts of Bihar, India.
The website highlights famous art forms like Madhubani paintings, Sikki grass craft, Lac bangles, Terracotta art, Tikuli art, and more while also introducing the artisans behind these crafts.

The goal of this project is to promote awareness of Bihar’s traditional crafts and provide a simple e-commerce style platform where users can explore products and learn about the cultural heritage of the region.

---

## Features

* Multi-page website design
* Responsive navigation bar with active page highlight
* Hero section with Bihar heritage background
* Products page showcasing various Bihar handicrafts
* Heritage page explaining the cultural background of crafts
* Artisan section introducing traditional craft makers
* Contact page with a user message form
* Login and Sign-Up page UI
* Cart and Checkout page layout
* Modern footer with useful links and contact information
